# Student Marks Analysis System

A Python-based data analysis tool that processes student performance data from a CSV file. It calculates key statistics and generates visual reports to help educators understand class performance at a glance.

##  Features
- **Data Loading**: Reads student names and subject marks from a CSV file.
- **Statistical Analysis**: Automatically calculates:
  - Average score per subject.
  - Maximum and Minimum scores per subject.
  - Total marks and ranking for each student.
- **Visualizations**: 
  - **Bar Chart**: A sorted ranking of student performance.
  - **Histogram**: A frequency distribution of class scores.
